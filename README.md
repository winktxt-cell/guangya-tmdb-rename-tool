# 🎬 光鸭网盘 TMDB 批量重命名助手

## ✨ 核心特性
- **极简设计**：砍掉了所有不必要的交互逻辑，包括撤销功能，主打一个点完即走。

- **高效纯净**：无多余权限请求，代码公开透明。

## 🔑 配置说明
本工具需要使用 TMDB API 才能正常工作。请按照以下步骤配置：

1. 登录 [TMDB 官网](https://www.themoviedb.org/settings/api) 申请你自己的 API Key。
2. 安装本脚本后，点击页面上的“设置”按钮（或首次运行时根据提示）。
3. 输入你的 API Key 并保存。

> **安全提示**：请勿将您的 API Key 泄露给他人。
